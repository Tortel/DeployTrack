Created because all the other Android alternatives looked terrible, this is a simple app intended to let you keep track of how much time is left for someone's deployment.
It is free and open source - all the code is available on GitHub, licensed under the Apache License, Version 2.

Libraries used:

* DateTimePicker
* HoloColorPicker
* HoloGraphLibrary
* Joda Time
* OrmLite
* Material Dialogs
* PagerSlidingTabStrip
